# IPL Bowlers Performance & Comparison Dashboard

## Project Overview

This project is an interactive IPL Bowlers Performance & Comparison Dashboard developed using Microsoft Power BI. The dashboard enables detailed analysis and comparison of IPL bowlers across multiple performance metrics from IPL 2008–2026 (up to Match 62 of the ongoing season).

The project was completed end-to-end in under 24 hours, covering data collection, transformation, modeling, DAX calculations, and dashboard design.

## Key Features

* Bowler vs Bowler comparison
* Economy Rate analysis
* Bowling Strike Rate analysis
* Total Wickets and Wicket Contribution %
* Team-wise bowling performance insights
* Interactive player search and slicers
* Dynamic KPI cards and visualizations
* Venue and match-based performance filtering

## Data Enrichment

To improve data quality and analytical depth, AI-assisted enrichment was performed using ChatGPT and Google Gemini for:

* Bowling Style classification
* Bowler Specialty categorization
* Ground Type identification
* Pitch Condition mapping across venues
* Performance Category standardization
